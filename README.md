# CameraSlewer

Application provide user an interface to create camera object and locations. Camera object will be mapped to nearest location

**Backend :**

Spring boot 

Embedded Mongo DB

Gradle 6.6


**Front End :**

Angular 10

Material UI

** Third Party Libraries : **

Google Maps


**Run Commands :**

Service : java -jar geomapper-1.0.0.jar

Angular UI : ng serve
